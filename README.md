# Deployment 101: Templates

This repository contains sample projects for the Deployment 101 workshop. It serves as a collection of templates for practicing different deployment strategies.

## Repository Structure

### Static Hosting
These projects are frontend-only and can be deployed to services like GitHub Pages, Vercel, or Netlify.
- **Cancer Detector**: A medical analysis dashboard interface.
- **NUV**: A multi-page university website template.
- **Rock-Paper-Scissor**: A browser-based game.
- **Weather App**: A weather forecasting dashboard.

### Managed Hosting
These projects require a build step or a server-side environment.
- **Geo-Globe-Three**: A modern Next.js application with an interactive 3D globe.
- **Simple Portfolio**: A simple server-side portfolio using Next.js

## Getting Started

1. Clone this repository.
2. Navigate to the specific project folder you want to deploy.
3. Follow the workshop instructions for your chosen deployment platform.
